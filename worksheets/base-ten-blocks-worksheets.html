Reading base-ten blocks (mixed)

Base-ten blocks (also called Dienes blocks) make place value physical: a unit cube is one, a rod is ten ones, a flat is a hundred ones. Counting a mixed pile — flats by hundreds, rods by tens, cubes by ones — is the bridge from seeing a number to writing it.

Questions parents & teachers ask

How do I teach base-ten blocks (mixed)?

Sort the pile by size first: flats, then rods, then cubes. Count each group in its own place and write the digits left to right. Mixing two- and three-digit numbers on one sheet stops kids guessing and makes them actually read the blocks. Print a fresh sheet, read a few piles together each day, and check with the answer key — short, regular practice beats one long session.

What grade are base-ten blocks (mixed) for?

This is usually taught in Grade 1 to Grade 3. Base-ten blocks (also called Dienes blocks) make place value physical: a unit cube is one, a rod is ten ones, a flat is a hundred ones. Counting a mixed pile — flats by hundreds, rods by tens, cubes by ones — is the bridge from seeing a number to writing it.

What does place value mean?

Place value is the rule that a digit's position gives its size: in 235 the 2 means two hundreds, the 3 means three tens and the 5 means five ones. Base-ten blocks make it concrete — a unit cube is one, a rod is ten ones, and a flat is a hundred ones — so kids can count the blocks and read off each digit.
