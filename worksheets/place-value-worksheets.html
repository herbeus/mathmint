Questions parents & teachers ask

What order should kids learn place value in?

Start with tens and ones — two-digit numbers shown as ten-rods and unit cubes — until "three tens and four ones is 34" is automatic. Add hundred-flats for three-digit numbers, counting hundreds, tens and ones in that order. Then mix two- and three-digit piles of base-ten blocks so kids have to read the blocks rather than guess. Each worksheet above sets exactly one of those stages.

What are base-ten blocks?

Base-ten blocks (or Dienes blocks) are a place-value model: a small unit cube is one, a rod is ten ones in a column, and a flat is a hundred ones in a 10×10 square. Building a number from blocks and then writing its digits is how kids connect the physical quantity to the way we write numbers.

What grade is place value taught in?

Tens and ones start in Grade 1; hundreds, tens and ones come in Grade 2; and mixed base-ten block work runs from Grade 1 through Grade 3. Pick the page that matches where your child is.
